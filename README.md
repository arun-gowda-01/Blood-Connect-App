# 🩸 Blood Connect

Blood Connect is a cross-platform Flutter application that connects blood donors with recipients during emergencies. The app allows users to register as donors, search for nearby donors, send emergency blood requests, and manage their profiles using Firebase Authentication and Cloud Firestore.

---

## ✨ Features

- 🔐 Google Sign-In Authentication
- 👤 User Profile Management
- 🩸 Blood Donor Registration
- 🔍 Search Donors by Blood Group
- 📍 Karnataka District-wise Donor Search
- 🚨 Emergency Blood Requests
- 📋 Donation History
- ☁️ Firebase Firestore Integration
- 📱 Responsive Flutter UI

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/arun-gowda-01/Blood-Connect-App.git
```

2. Navigate to the project

```bash
cd Blood-Connect-App
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
